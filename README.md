# GUI-DNS-changer-for-linux
simple GUI DNS changer for linux with PyQt5

tested with linux "mint and ubuntu"

### Installation
`git clone https://github.com/mrmoein/GUI-DNS-changer-for-linux.git`

`cd GUI-DNS-changer-for-linux/`

`pip3 install PyQt5`

`sudo python3 dns.py`
